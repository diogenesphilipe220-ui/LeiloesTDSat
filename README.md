\# LeiloesTDSat



\## Sobre o Projeto

Sistema de gerenciamento de leilões desenvolvido em Java com 

interface gráfica desktop (Swing/JFrame). Permite cadastrar 

e listar produtos disponíveis para leilão com persistência 

de dados em banco MySQL.



\## Tecnologias Utilizadas

\- Java (NetBeans IDE)

\- MySQL

\- JDBC

\- Git e GitHub
